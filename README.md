Developed a beginner-friendly educational website using HTML, CSS, and JavaScript, designed to help students access learning material easily. The website includes dedicated pages for Notes, Quizzes, and Activities, making learning interactive and engaging. Implemented a clean and responsive layout so the site works smoothly on mobile and desktop devices.
The project connects teaching concepts with frontend development by providing structured content, simple navigation, and an interactive quiz section.

Key Features:

Home page with quick navigation to notes, quizzes, and about page

Notes page displaying subject-wise learning material

Quiz page with basic JavaScript-based Q&A

Fully responsive design

Organized folder structure with assets for images and notes

Tech Stack:
HTML, CSS, JavaScript
